pub async fn get_all_users(__arg0: Extension<Arc<PgPool>>) -> impl IntoResponse
Expand description

Récupère tous les utilisateurs de la base de données.

Route : GET /users/all

Réponse :

  • Succès (200 OK) : Liste des utilisateurs avec leurs user_id, username et email.
    [
  {
    "user_id": "123",
    "username": "user1",
    "email": "user1@example.com"
  },
  ...
]
  • Erreur (500 Internal Server Error) : Si la récupération des utilisateurs échoue.
    {
  "error": "Failed to retrieve users"
}
